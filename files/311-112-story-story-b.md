# Story B

## User Story

As a user, I want to move a document into a different folder so that I can correct or customise the generated organisation.

## Acceptance Criteria

- A file can be moved from one folder to another.
- The updated location is reflected immediately in the UI.
- The updated location remains after reloading the app.

## Edge Cases

- Moving into the same folder should not create duplicates.
- Moving a missing file should show a clear error.
